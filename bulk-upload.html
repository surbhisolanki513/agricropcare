CSV Format Guidelines:

  • Type: Either INWARD or OUTWARD.
  • ClientName: Must match an existing client in the system.
  • CommodityName: Must match an existing commodity name.
  • WarehouseName: Must match an existing warehouse name.
  • QuantityMT: Quantity in metric tons (e.g. 22.730).
  • Date: Transaction date in YYYY-MM-DD format (e.g. 2026-06-30).
